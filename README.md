# StockSim – Virtual Portfolio Simulator

A web-based stock market simulator that allows users to practice trading with virtual money. The application provides a real-time-like experience of buying and selling stocks while tracking portfolio performance.

## 🚀 Features
- Search and select stocks from NSE, NASDAQ, and NYSE
- Buy and sell stocks with virtual cash
- Real-time simulated stock price updates
- Portfolio tracking with profit/loss calculation
- Interactive portfolio value visualization using charts
- Trade history log for all transactions
- Predefined stock suggestions for quick trading

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Chart.js

## 📊 Functional Highlights
- Dynamic state management for portfolio, transactions, and cash balance
- Simulated live stock pricing with random market fluctuations
- Data visualization for portfolio growth over time
- Responsive and interactive UI for smooth user experience

## 💻 How to Run
1. Download or clone the repository
2. Open `index.html` in any modern browser
